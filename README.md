## Installation

1. Create .env file
2. Copy conent .env.example to .env
3. setting your database config in .env

```bash
$ run `npm install`
$ Make sure you connect database success
$ run `npx sequelize-cli db:migrate` to create table users in database
$ Site: http://localhost:7000/
```
